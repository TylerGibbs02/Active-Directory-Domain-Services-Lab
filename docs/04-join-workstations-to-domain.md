# Join Client to Domain

<br>

## Objectives
Join Windows 11 VM client to the domain

<br>

## Steps taken
- Prestaged computer 'Client01' in DC
- Change Windows 11 VM name to Client01 and assign IP and DNS settings
- Change from Workgroup to lab.local domain and sign in through administrator credentials

<br>

## Validation

Windows 11 VM joined to lab.local domain
