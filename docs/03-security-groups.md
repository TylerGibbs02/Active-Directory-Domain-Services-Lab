# Security Groups

<br>

## Objectives
Create security groups for students, faculty, and staff to later be tied to GPOs for computer access restrictions

<br>

## Steps Taken
- Created groups:
  - Student_ClassroomPC_access
  - Faculty_StaffPC_access
  - IT_Helpdesk
- Assign Students to Student_ClassroomPC_access
- Assign Faculty to Faculty_StaffPC_access
- Assign IT_Staff to IT_Helpdesk
